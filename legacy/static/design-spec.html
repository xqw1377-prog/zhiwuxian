04. 动效哲学 / Motion Language

  • ✦ 线性渐进： 所有线条的出现应使用 cubic-bezier(0.4, 0, 0.2, 1) 缓动函数。
  • ✦ 呼吸感： 关键 UI 元素可带有微弱的渐变呼吸效果（Opacity 0.3 - 0.7）。
  • ✦ 极轻量： 交互反馈不应有沉重的位移，仅通过线条变亮或缩放（1.02x）体现。
